# Getting Stated

* clone

```
git clone ...
cd node-react-tutorial
```

* npm install

```
npm install -g bower browserify exoress express-generator gulp node-inspector nodemon
npm install
```

* run

```
gulp
```

* see http://localhost:8080 with your browser
